非零环绕规则

渐变

    画笔属性
  1. lineWidth:线宽，默认1px 若想细且清晰 +-0.5
  2. lineCap:线末端类型 butt默认、round、square
  3. lineJoin:相交线的拐点 miter默认、round、bevel
  4. strokeStyle:线的颜色
  5. fillStyle:填充颜色
  6. setLineDash:设置虚线 setLineDash([5,10,15])虚实交替
  7. getDashOffset:设置虚线偏移量（负值向右）getDashOffset=-20

网格

    绘制矩形
  1. rect(x,y,w,h)无独立路经
  2. strokeRect(x,y,w,h)有独立路经，不影响别的绘制
  3. fillRect(x,y,w,h)有独立路经，不影响别的绘制
  4. clearRect(x,y,w,h)擦除矩形区域
    绘制圆弧
  1. arc( centerx,centery,radius,startingAngle,endingAngle,anticlockwise=false)
  2. strokeRect(x,y,w,h)有独立路经，不影响别的绘制
  3. fillRect(x,y,w,h)有独立路经，不影响别的绘制
  4. clearRect(x,y,w,h)擦除矩形区域
